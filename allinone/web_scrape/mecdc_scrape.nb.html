Maine COVID19 By County

The Maine CDC COVID19 information site is a wonderful resource for getting updated info. The site provides a number of tables with counts of cases by county, age and sex. The site also provides a map of cases by county which is interesting, but misses the importantance normalizing counts by population. This note attempts to augment the state case by county map with a case density by county map companion using tidyverse tools (and friends).

We first retrieve the most recent census info (10 years old now) from the Maine State economist’s office. The office published spreadsheets of population by county

Like many HTML tables, the table itself is messy - great for reading online but annoying to parse for computation. Through trial and error the correct table identified (it’s currently the third table on the page.)

tbl <- MeCDC %>%

  .[[3]] %>%
  rvest::html_table()

Uggg. The column names are repeats of the table name, and the first row has the actual column names.

So, we will do some back flips to rename the column. Then we will convert the counts columns to numerics, followed by ordering alphabetically and merging in the population counts. Finally, we add a density column (per 1000 people) using a function - each new column name is prepended with “d” for density.

colnames(tbl) <- gsub("[0-9]", "", unname(as.vector(tbl[1,])))
x <- tbl[-1,] %>%
  dplyr::as_tibble() %>%
  dplyr::mutate_at(colnames(tbl)[-1], as.numeric) %>%
  dplyr::arrange(County) %>%
  dplyr::left_join(MePop, by = "County") %>%
  dplyr::mutate(
    dConfirmed = Confirmed/(Pop2010/1000),
    dRecovered = Recovered/(Pop2010/1000),
    dHospitalizations = Hospitalizations/(Pop2010/1000),
    dDeaths  = Deaths/(Pop2010/1000)
  )
x
